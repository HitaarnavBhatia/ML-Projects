# 🥇 Olympic Medal Prediction – Beginner ML Project

## Project Overview

This is my first end-to-end machine learning project.

The goal of this project was to understand the complete ML workflow — from forming a hypothesis to training a model and evaluating its performance.

Using historical Olympic data, I built a machine learning model to predict how many medals a country might win based on past performance and team characteristics.

---

#  Problem Statement

Can we predict a country’s Olympic medal count using:

- Number of athletes  
- Previous medal counts  
- Historical Olympic data  

---

#  Machine Learning Project Steps

Most machine learning projects follow a structured workflow.  
In this project, I followed these steps:

**Project Steps**

1. Form a hypothesis  
2. Explore and understand the data  
3. Clean and prepare the dataset  
4. Select an error metric  
5. Split the data into training and testing sets  
6. Train a Linear Regression model  
7. Evaluate model performance  

---

##  Model Used

- **Linear Regression (scikit-learn)**

The model was trained to predict medal counts using:

- Number of athletes  
- Previous medal counts  

Model performance was evaluated using error analysis techniques to understand prediction accuracy.
